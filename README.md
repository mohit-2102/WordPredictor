# WordPredictor

AI-powered next word prediction web application built using LSTM architecture, Python, and Streamlit.

The model is trained on a custom text dataset to predict the most probable next word based on user input.

---

## 🚀 Features

- Next word prediction using LSTM neural networks
- Interactive Streamlit web interface
- Real-time text prediction
- Tokenization and sequence processing
- Custom trained NLP model
- Clean and responsive UI

---

## 🧠 Tech Stack

- Python
- TensorFlow / Keras
- LSTM
- Streamlit
- NumPy
- Pickle
- NLP

---

## ⚙️ How It Works

1. Text data is cleaned and tokenized
2. Sequences are generated from the dataset
3. An LSTM model is trained on sequence patterns
4. The trained model predicts the next probable word
5. Streamlit provides an interactive frontend for inference

## ▶️ Run Locally

Clone the repository:

```bash
git clone https://github.com/mohit-2102/WordPredictor.git
```

Move into the project folder:

```bash
cd WordPredictor
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the Streamlit app:

```bash
streamlit run app.py
```
